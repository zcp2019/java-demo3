【注意事项】

2.java类的说明。
 路径：MyDemoJava\src\com\oio\util
 SignDigest.java 此类为进行MD5加密的工具类。
 V2**Sub.java  此类为AX/AXB/AXYB的绑定接口类
 V2**UnSub.java 此类为AX/AXB/AXYB的解绑接口类。
 MyHttpClient.java 此类为绑定解绑调用的接口类
 UUIDGenerator.java 此类为随机生成请求业务id的工具类。
